## 개요

### 프로젝트 진행 기간

- 프로젝트 이름: 피유(Piyou)
- 프로젝트 기간: 2023.10.4 ~ 2023.11. SSAFY 9기 2학기 자율 프로젝트
- 개발 엔진 및 언어: Unity & C#
- 멤버: 팀장 최하영(권소정, 이다영, 이지영, 인영교, 정민)

## 기획 배경

올바른 식습관과 양치습관을 학습시켜 부모님의 육아 스트레스를 경감시키는 어플리케이션
